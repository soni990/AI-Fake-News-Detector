
AI Fake News Detector 📰🔍

Overview
AI Fake News Detector is a web application that analyzes news articles and determines whether they are likely to be real or fake using Artificial Intelligence and Natural Language Processing (NLP) techniques.

Features
•	Analyze news articles from URLs
•	AI-powered fake news detection
•	User-friendly interface
•	Fast and accurate content analysis
•	Detailed analysis results

Technologies Used
•	Python
•	Streamlit
•	Google Gemini API
•	Scikit-learn
•	BeautifulSoup

How It Works
1.	User enters a news article URL.
2.	The application extracts the article content.
3.	The content is processed and sent to the AI model.
4.	The AI analyzes the article and generates a credibility assessment.
5.	Results are displayed to the user.

Environment Variables
Create a .env file and add:
GEMINI_API_KEY=your_api_key_here

Run the Project
streamlit run app.py
or
python app.py
